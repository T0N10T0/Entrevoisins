
# Entrevoisins

### Description

Entrevoisins est une application qui permet à des personnes d’un même quartier de se rendre des petits services : garde d’animaux, petit bricolage, troc d’objets, cours particuliers...

**Fonctionnalités présentes :**
* lister mes voisins ;
* ajouter un voisin ;
* suppression d’un voisin.
 
**Fonctionnalités ajoutées :**
* nouvel écran de détails de l'utilisateur (avatar et nom) avec bouton de retour ;
* bouton d'ajout du voisin à la liste de favoris ;
* onglet Favoris affichant les utilisateurs marqués comme favoris.

### Exécuter et compiler le projet avec Android Studio

1. Cloner ou télécharger les sources du projet depuis : https://github.com/T0N10T0/Entrevoisins
2. Ouvrir le projet avec **Android Studio** :
* Pour exécuter le projet : sélectionner un périphérique (physique ou virtuel) et cliquer sur le bouton 'Run' (Maj+F10*).
* Pour compiler le projet : cliquer sur le bouton 'Make Project' (Ctrl+F9*).

**sous Windows*
